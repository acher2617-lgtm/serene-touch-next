# Serene Touch Next.js Landing Page

One-page Next.js landing page for a premium massage and wellness studio.

## Stack
- Next.js 15
- React 19
- TypeScript
- Custom CSS via `app/globals.css`

## Notes
- Tailwind is intentionally **not** used in this project.
- Styling is fully handled with custom CSS.
- Use only `next.config.mjs` in the project root.

## Local development
```bash
npm install
npm run dev
```

## Production build
```bash
npm install
npm run build
```

## Netlify
This project already includes `netlify.toml` configured for the Netlify Next.js plugin.

## Deploy
1. Push the project to GitHub
2. Import the repository into Netlify
3. Netlify should detect the Next.js setup automatically
4. Deploy
